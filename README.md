# Resume Webpage (HTML Only)

This is a beginner-friendly resume webpage project created using only HTML. The purpose of this project was to practice core HTML concepts such as semantic tags, headings, paragraphs, lists, links, and page structure while building a simple personal resume website.

## Features

* Clean resume webpage structure
* About Me, Education, Skills, Projects, and Contact sections
* Semantic HTML layout
* Beginner-friendly and easy to understand

## Technologies Used

* HTML5

## Purpose of This Project

This project is part of my web development learning journey. I created it to strengthen my HTML fundamentals before moving on to CSS, JavaScript, and MERN Stack development.
